Información de la Práctica

La práctica consta de las siguietes partes

- Al momento de cargar la pagina si hay algún registro en el localStorage se mostrarán dichos registos en la parte de "Lista de personas"

- Si no hay ningún registro en el localStorage, se podrá cargar los registros desde el computador, eligiendo un archivo .json

- Se podrá agregar nuevos registros de personas, estos se mantendrán en el loscalStorage, en caso de querer guardar los registros en un archivo jason se debera presionar el bot´pn guardar.

Botones:

  • Agregar Persona: Este botón es para agregar a una persona. Este nuevo registro estará en el localStorage mas no en el archivo .json
  • Guardar Datos: Este botón es para guardar todos los registros que se encuentran en el localStorage a un archivo .json externo, puedo crear un nuevo documento .json o reemplazar el archivo "personas.json" que ya existe en la carpeta del proyecto.
  • Cargar Datos: Botón útil para cargar los datos desde un archivo .json externo al localStorage, una vez los datos estén en el localStorage se imprimirán en la pantalla del navegador.
  • Eliminar: Este botón está alado de cada registro para poder eliminarlo.
  • Modificar: Este botón modifica un registro existente.
