Información del Proyecto

Este es un proyecto de Mantenimiento de Personas. Puede agregar, eliminar, guardar y cargar personas usando localStorage y archivos JSON.

  • Agregar Persona: Complete el formulario y haga clic en "Agregar Persona".
  • Eliminar Persona: Haga clic en "Eliminar" junto a la persona correspondiente.
  • Guardar Datos: Haga clic en "Guardar Datos" para descargar un archivo JSON con las personas.
  • Cargar Datos: Haga clic en "Cargar Datos" y seleccione un archivo JSON para cargar personas.
