Detalles de la Práctica

- Al momento de cargar la pagina lo primero que se muestra es un formulario para agregar una persona nueva."

- Antes de agregar la persona se deberá escoger si queire que esta sea almacenada en localStorage ó sessionStorage

- Otra opción tambien es cargar registros desde un archivo .json (existe uno en la carpeta del proyecto llamado "personas.json").

Botones:

  • Agregar Persona: Este botón es para agregar a una persona. Este nuevo registro estará en el localStorage mas no en el archivo .json
  • Cargar Datos: Botón útil para cargar los datos desde un archivo .json externo, cuando se carguen los datos estos de guradarán en el localStorage, y si es que hay otros registros en el localStorage se sobreescribirán.
  • Guardar Datos: Este botón es para guardar todos los registros que se encuentran en el localStorage a un archivo .json externo.
  • Mostrar LocalStorage: Este botón imprime en pantalla lo que se encuentra en el momento actual en el localStorage.
  • Mostrar SessionStorage: Este botón imprime en pantalla lo que se encuentra en el momento actual en el sessionStorage.
  • Eliminar: Este botón está alado de cada registro para poder eliminarlo.
  • Modificar: Este botón modifica un registro existente.

Diferencias entre el uso de localStorage y sessionStorage:

Cuándo uso localStorage y cierro la pestaña o ventana de la pagina al volver a abrir la pagina los registros de las personas se mantienen tal cual estaban antes de cerrar.
